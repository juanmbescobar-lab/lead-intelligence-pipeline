# Runbook

Operational procedures for the Lead Intelligence Pipeline.

## Services

| Service | Port | URL |
|---------|------|-----|
| n8n | 5678 | http://EC2_IP:5678 |

## Common Operations

### Start all services
```bash
make up
```

### Stop all services
```bash
make down
```

### View logs
```bash
make logs
```

### Health check
```bash
make health
```
